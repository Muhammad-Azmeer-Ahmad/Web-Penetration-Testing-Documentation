## Quick Notes So Far
- Web apps run client-server architecture — front end in browser,
  back end on server
- Unlike static websites, web apps are dynamic and change per user
- No installation needed — runs in browser, updates server-side only
- This is what makes them powerful and also what makes them risky

## Key Attack Surface Insight
Web apps are accessible by anyone with a browser and internet
connection. A single code change can introduce a critical vuln.
SQL injection, file upload, IDOR, broken access control — all
covered later in this path.
